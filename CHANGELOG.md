# Changelog

## 0.3.0 — 2026-03-27

* Virtualised elements pane scrolling to fix canvas lag on large files

## 0.2.0 — 2026-03-27

* Added Makefile with `bundle`, `bundle-patch`, `bundle-minor`, `bundle-major` targets
* Added Filter pane with element type and path style (fill/stroke) visibility toggles
* Added automatic element filtering for large SVGs with on-screen phase timing log
* Added macOS app bundle configuration with icon
* Added loading spinners and vertex budgets for large files
* Added open-from-URL support
* Added clip-path handling
* Added indexed spacebar hover search for inspecting elements on canvas

## 0.1.0 — 2026-03-09

* Initial release — basic SVG renderer with element tree pane and GPU-accelerated canvas
